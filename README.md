# Trangle Tracker

#### By Katarina Tuttle

## Description

This web application will tell user if triangle is Right, Equilateral, Isosceles or Scalene.

## Technologies Used
* HTML
* CSS
* Bootstrap
* Ruby

## Setup/Installation Requirements
* Clone and navigate to repository in Terminal
* Run commands:
  * $ ruby app.rb
